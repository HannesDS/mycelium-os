## ADDED Requirements

### Requirement: Website agency shrooms

The system SHALL include shroom definitions for a static frontend website agency: front-end engineer, QA engineer, and sales/marketer. Each SHALL have appropriate skills and tools for their role.

#### Scenario: Front-end engineer shroom

- **GIVEN** the front-end engineer shroom
- **WHEN** it is invoked
- **THEN** it SHALL have access to programming tools (e.g. web_browser, code execution or MCP), repo access (read), and skills for building static web frontends

#### Scenario: QA engineer shroom

- **GIVEN** the QA engineer shroom
- **WHEN** it is invoked
- **THEN** it SHALL have skills for testing (functional, best practices) and SHALL be able to validate deliverables

#### Scenario: Sales/marketer shroom

- **GIVEN** the sales/marketer shroom
- **WHEN** it is invoked
- **THEN** it SHALL have email skills, intake requirements handling, and SHALL be able to send/receive from a configured client mailbox
